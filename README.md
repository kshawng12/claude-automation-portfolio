# Claude AI Automation Portfolio

I build automation pipelines that use Claude AI to handle the work that takes you hours — researching, writing, formatting, and sending — so you can focus on running your business.

## How It Works

Every automation follows the same reliable pattern:

```
1. You provide the inputs (topic, data, URL, etc.)
2. Claude reasons through the task
3. Python scripts execute each step deterministically
4. You get a finished, ready-to-use output
```

No hallucination loops. No fragile one-shot prompts. Each step is tested, documented, and recoverable.
